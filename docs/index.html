BRIM Live Reservoir Feed

This GitHub Pages site hosts a small near-live GeoJSON feed for BRIM reservoir storage.

Main feed:

  • data/cdec_reservoir_latest.geojson
  • data/cdec_reservoir_latest_summary.json

Data are provisional and sourced from the California Data Exchange Center (CDEC). BRIM should use the feed and observation timestamps to warn when data are stale.
